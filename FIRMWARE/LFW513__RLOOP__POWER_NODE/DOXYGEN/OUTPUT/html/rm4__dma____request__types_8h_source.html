18 4.16.2 Default DMA Request Map
19 The DMA module on this microcontroller has 16 channels and up to 32 hardware DMA requests. The
20 module contains DREQASIx registers which are used to map the DMA requests to the DMA channels. By
21 default, channel 0 is mapped to request 0, channel 1 to request 1, and so on.
22 Some DMA requests have multiple sources, as shown in Table 4-32. The application must ensure that
23 only one of these DMA request sources is enabled at any time.
