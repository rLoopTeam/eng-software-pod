enum mpuRegionAccessPermission

Alias names for mpu region access permissions.

This enumeration is used to provide alias names for the mpu region access permission:

  • MPU_PRIV_NA_USER_NA_EXEC no access in privileged u32Mode, no access in user u32Mode and execute
  • MPU_PRIV_RW_USER_NA_EXEC read/write in privileged u32Mode, no access in user u32Mode and execute
  • MPU_PRIV_RW_USER_RO_EXEC read/write in privileged u32Mode, read only in user u32Mode and execute
  • MPU_PRIV_RW_USER_RW_EXEC read/write in privileged u32Mode, read/write in user u32Mode and execute
  • MPU_PRIV_RO_USER_NA_EXEC read only in privileged u32Mode, no access in user u32Mode and execute
  • MPU_PRIV_RO_USER_RO_EXEC read only in privileged u32Mode, read only in user u32Mode and execute
  • MPU_PRIV_NA_USER_NA_NOEXEC no access in privileged u32Mode, no access in user u32Mode and no execution
  • MPU_PRIV_RW_USER_NA_NOEXEC read/write in privileged u32Mode, no access in user u32Mode and no execution
  • MPU_PRIV_RW_USER_RO_NOEXEC read/write in privileged u32Mode, read only in user u32Mode and no execution
  • MPU_PRIV_RW_USER_RW_NOEXEC read/write in privileged u32Mode, read/write in user u32Mode and no execution
  • MPU_PRIV_RO_USER_NA_NOEXEC read only in privileged u32Mode, no access in user u32Mode and no execution
