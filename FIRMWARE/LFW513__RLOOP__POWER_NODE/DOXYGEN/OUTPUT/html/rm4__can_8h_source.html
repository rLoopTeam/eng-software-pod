84  /*****************************************************************************
85  when configuring the message object, set the direction (bit 29 of the IF)
86  Message direction
87  0 Direction = Receive: On TxRqst, a Remote Frame with the identifier of this message object is
88  transmitted. On receiving a Data Frame with a matching identifier, this message is stored in this
89  message object.
90  1 Direction = Transmit: On TxRqst, the respective message object is transmitted as a Data
