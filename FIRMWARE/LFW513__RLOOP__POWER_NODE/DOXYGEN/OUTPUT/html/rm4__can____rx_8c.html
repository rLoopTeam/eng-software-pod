Checks if a Message has arrived to a specific Message Object The bit corresponding to the MessageBox NewDat.

Datasheet: 23.17.11 New Data X Register (DCAN NWDAT X) With the New Data X Register, the CPU can detect if one or more bits in the different New Data Registers are set. Each register bit represents a group of eight message objects. If at least on of the NewDat bits of these message objects are set, the corresponding bit in the New Data X Register will be set.
