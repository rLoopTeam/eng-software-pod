176  /**

178  * Command Code 78h
179  * Command Parameter Direction Byte
180  * Description Generates three time slots: two read time slots and one write time slot at the 1-Wire line. The type
181  * of write time slot depends on the result of the read time slots and the direction byte. The direction
182  * byte determines the type of write time slot if both read time slots are 0 (a typical case). In this
183  * case, the DS2482-100 generates a write-one time slot if V = 1 and a write-zero time slot if V = 0.

185  * * If the read time slots are 0 and 1, they are followed by a write-zero time slot.
186  * * If the read time slots are 1 and 0, they are followed by a write-one time slot.
187  * * If the read time slots are both 1 (error case), the subsequent write time slot is a write-one.
188  * Typical Use To perform a 1-Wire Search ROM sequence; a full sequence requires this command to be
189  * executed 64 times to identify and address one device.
190  * Restriction 1-Wire activity must have ended before the DS2482-100 can process this command.
191  * Error Response Command code and direction byte is not acknowledged if 1WB = 1 at the time the command
192  * code is received and the command is ignored.
193  * Command Duration 3 x tSLOT + maximum 262.5ns, counted from the falling SCL edge of the first bit (MSB) of the

196  * Read Pointer Position Status Register (for busy polling).
